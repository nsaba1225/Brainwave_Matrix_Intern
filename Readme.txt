ToDo-list web-app is simple and created to make you remember tasks and complete it at your desired time.
This web-app helps you to create a task with specific date, type(name of the task) and time.
You can check(once completed), update, and delete the task.
Programming languages used are: HTML, CSS, JAVASCRIPT.
Editor: Vscode
Create Index.hmtl file 
then Css file(Style.css) for styling
and script.js: javascript file
